### Setup du project
1. Ouvrir une fenetre "Git Bash" et y indiquer la commande suivante:
> ./dev setup

### Docker
1. Start service and database
> ./dev start
2. Stop service and database
> ./dev stop
3. Remove service and database
> ./dev rm
